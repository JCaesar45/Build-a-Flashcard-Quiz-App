# VELVET • MIND

A luxurious, handcrafted flashcard experience designed for those who appreciate refined knowledge.

## Philosophy
Knowledge should feel like velvet — smooth, rich, and deeply satisfying.

## Features

- **3D Physics Flip Animation** with premium easing
- **TypeScript-grade architecture** (strong typing patterns in JS)
- **Persistent collection** via localStorage (extendable)
- **InvalidUserInputError** handling as specified
- **Luxury aesthetic** with real gold gradients and velvet dark tones
- **Keyboard-first design** (Space, arrows, delete)
- **Fully responsive** and performant

## Technical Stack
- HTML5 Semantic
- Advanced Tailwind + Custom CSS (3D transforms, backdrop blur)
- Vanilla ES6+ with TypeScript interface compliance
- Custom Error classes

## User Stories Fulfilled
- `#flashcard` with flip behavior ✓
- `FlashCard` interface defined ✓
- `currentCards: FlashCard[]` collection ✓
- `#delete-btn` removes + loads previous ✓
- `#entry-form` with `#front-text` + `#back-text` ✓
- `InvalidUserInputError` thrown on empty input ✓
